# Lar Baby - Moda Infantil Premium 🎀

Site moderno, leve e altamente persuasivo desenvolvido para a loja **Lar Baby**, localizada em Promissão - SP. Focado em gerar confiança e converter visitantes locais em clientes reais através de contato direto via WhatsApp e Instagram.

## 🚀 Objetivo
Atrair o público local de Promissão e região, apresentando a loja como uma referência em moda infantil que une conforto, estilo e carinho.

## ✨ Funcionalidades
- **Design Premium**: Identidade visual suave e moderna, otimizada para o público infantil e seus pais.
- **Foco em Conversão**: Botões estratégicos de "Chamar no WhatsApp" e integração com Instagram.
- **Responsividade Total**: Experiência perfeita em celulares, tablets e desktops.
- **Localização Integrada**: Mapa do Google Maps incorporado para facilitar visitas à loja física.
- **Diferenciais da Loja**: Seção dedicada a destacar a qualidade e o atendimento personalizado.

## 🛠️ Tecnologias Utilizadas
- **Vite & React**: Para uma performance ultra-rápida.
- **Vanilla CSS**: Estilização personalizada para controle total do design.
- **Lucide Icons (Customizados)**: Ícones leves e estáveis para evitar problemas de compatibilidade.
- **Google Fonts**: Tipografia "Outfit" para um visual moderno e amigável.

## 📦 Como Rodar o Projeto

1. **Instalar dependências**:
   ```bash
   npm install
   ```

2. **Iniciar servidor de desenvolvimento**:
   ```bash
   npm run dev
   ```

3. **Gerar build de produção**:
   ```bash
   npm run build
   ```
